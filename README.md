# 📦 Distributor Manager — Tier 4: Artifact Release Gateway

Welcome to the **distributor-manager**. This repository serves as the Tier 4 Release Gateway. It is responsible for catching compiled artifacts from the `container-manager` (Builder) and publishing them securely as GitHub Releases.

* **Global Systems Cockpit:** [github-manager](https://github.com/RPDevs-Vault/github-manager) (Live Health & Fleet Runners)
* **Active Project Roadmap:** [project-manager](https://github.com/RPDevs-Vault/project-manager) (Task & Issue Sync)
