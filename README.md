# 📦 Distributor Manager — Tier 4: Artifact Release Gateway

Welcome to the **distributor-manager**. This repository serves as the Tier 4 Release Gateway. It is responsible for catching compiled artifacts from the `container-manager` (Builder) and publishing them securely as GitHub Releases.

* **Tier 0 Global Cockpit:** [github-manager](https://github.com/RPDevs-Vault/github-manager) (Live Health & Fleet Runners)
* **Tier 1 Governance Hub:** [vault-manager](https://github.com/RPDevs-Vault/vault-manager) (Policies & Dispatching)
* **Tier 2 Builder Fleet:** [container-manager](https://github.com/RPDevs-Vault/container-manager) (CI/CD & Registries)
* **Tier 3 Project Roadmap:** [project-manager](https://github.com/RPDevs-Vault/project-manager) (Task & Issue Sync)
